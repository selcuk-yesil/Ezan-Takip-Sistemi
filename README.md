# Ezan-Takip-Sistemi
Desktop Prayer Times Tracker | Python-based real-time notification system with API integration. Features system tray support and multi-threaded UI for seamless performance. / API entegrasyonlu, çoklu iş parçacığı (threading) mimarisiyle çalışan, sistem tepsisi destekli masaüstü ezan vakti takip ve bildirim uygulaması.
